# 🏛️ Zênite Acadêmico | Gestão de Elite Nacional

Landing Page de alta conversão para assessoria acadêmica premium, com foco em UI/UX sofisticado e gatilhos de escassez.

## 💎 Diferenciais
* **Fundo Dinâmico:** Efeito estelar em Canvas 2D.
* **Gatilhos de Conversão:** Cronômetro de sessão e notificações de vendas em tempo real.
* **Design Premium:** Estética Dark com detalhes em Dourado (#D4AF37).
* **Mobile First:** Totalmente responsivo para WhatsApp.

## 🛠️ Tecnologias
* HTML5 / CSS3 / JavaScript Vanilla

O projeto pode ser visualizado diretamente pelo **GitHub Pages** após a ativação nas configurações do repositório.
